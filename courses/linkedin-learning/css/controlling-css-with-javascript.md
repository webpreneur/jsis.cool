# Accessing Style Sheets and Styles

```js
// Access all style sheets in a document.
document.styleSheets;
```

<mark>Tip:</mark>

In order to disable styleSheet, set its `disabled` property to `true`.
It is useful for:
- debugging
- A/B testing
- feature toggling
